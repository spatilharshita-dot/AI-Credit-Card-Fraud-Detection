
# 💳 AI-Based Fraud Detection System

## 🔍 Project Description
This project detects fraudulent credit card transactions using Machine Learning (Logistic Regression).

## 🚀 Features
- Real-time fraud prediction
- Banking-style dashboard UI
- Fraud vs Legit demo mode
- Confusion matrix visualization
- Scaled ML pipeline

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## 📊 Model
- Logistic Regression
- StandardScaler used for preprocessing
- Trained on credit card dataset

## ▶️ How to Run
streamlit run app.py
